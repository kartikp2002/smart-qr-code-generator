# Smart QR Code Generator

A Flask-based web application to generate QR codes from text or URLs.

## Features
- User authentication
- QR code generation
- QR history
- Download QR image

## Technologies Used
- Python
- Flask
- SQLite
- Bootstrap

## How to Run
pip install -r requirements.txt
python app.py
